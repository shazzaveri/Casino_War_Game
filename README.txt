This is basic implementation for casino war game assignment.

The GRASP principle with MVC implementations is used between classes TableController, WarGame and Deck
The Loose coupling is implemented in seperate classes, like variation one , two and three. If amend anything in variation 1 , it wont affect variation 2 or 3.
Indirection is used in the variation controller classes by using the objects from classes Deck, Person.
Single Responsibilty priciple is followed such as each variation is focusing on one single logic to make the code manageable and focused.
